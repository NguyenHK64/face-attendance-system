# Face Attendance System (C# WinForms + OpenCV)

## 📌 Overview
A desktop application for employee attendance using face recognition.

## 🚀 Features
- Face detection using Haar Cascade
- Face recognition using LBPH algorithm
- Auto attendance logging
- Working hours calculation
- SQLite database storage

## 🛠 Tech Stack
- C# (.NET 8 WinForms)
- EmguCV (OpenCV wrapper)
- SQLite
